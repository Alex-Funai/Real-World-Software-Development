## Getting Started

*	Clone the repository to a local directory.
*	Install any missing Prerequisites, if you don't have them.
*	Check '2_DocumentManagementSystem\build\lib" for .jar that includes library/module dependencies.

### Prerequisites

* 	IntelliJ IDEA
* 	GIT BASH
* 	JDK-8.0(1.8)
*	Gradle-8.6.3
* 	Groovy-3.0.7
*	JUnit-4.13.2
*	Hamcrest-All-2.2


### Installing

1.	Clone the '2_DocumentManagementSystem' repository, or download package as a '.zip' locally.
2.	Open the repository as a project in IntelliJ.
3.	Use GraddleWrapper to build the project using the included '2_DocumentManagementSystem.jar'
